# 五类图说明

## bar

适合做比较。

重点关注：

- 单柱、分组柱、堆叠柱
- 是否从 0 开始
- 是否显示柱顶值
- 是否需要 hatch 强调主方法

## donut

适合做简单组成。

重点关注：

- 切片不要太多
- 最好直接标值或百分比
- 不适合做精确比较

## line

适合做趋势。

重点关注：

- 是否有 marker
- 是否有 confidence band
- 是否有 baseline / reference line
- 是否需要平滑

## radar

适合做少量维度的 profile 对比。

重点关注：

- 是否按轴归一化
- 标签是否过多
- 系列数量不能太多

## scatter

适合做相关性、分布、聚类。

重点关注：

- 点大小
- alpha
- 类别颜色
- 是否需要注释框
- 是否存在过度重叠
