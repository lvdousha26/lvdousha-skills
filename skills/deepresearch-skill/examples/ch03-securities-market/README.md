# Chapter 3 Securities Market Example

这是 `deepresearch-skill` 的一个真实交付样例。

任务目标是阅读《投资学精要（第九版）》第 3 章《证券市场》，基于教材正文、图表、公式、案例和必要的官方补充资料，产出一份中文 tutorial 风格的研究报告，并直接交付原生 LaTeX 与渲染后的 PDF。

## 包含文件

- `prompt.md`: 生成这份报告时使用的原始 prompt
- `report.pdf`: 最终渲染结果
- `screenshots/`: 从 PDF 中截出的代表页，供 README 展示

## 这个例子展示什么

- 章节级结构化重写，而不是 OCR 文本转存
- 图表、公式、表格和文字解释的混合写法
- 教材内容与外部时效性补充的边界处理
- 来源台账和勘误记录如何进入最终交付物

如果你只想看最终效果，直接打开 [report.pdf](./report.pdf)。
